## How to Add

Simply place your PDF resume file in this folder:
```
assets/resume/Rahul-Garg-Resume.pdf
```

The portfolio website has two download buttons:
1. In the navigation bar (top right)
2. In the Home/Hero section

Both will trigger the download of this file when clicked.

## Alternative: Google Drive Link

If you prefer to host your resume on Google Drive:

1. Upload resume to Google Drive
2. Right-click and select "Get link"
3. Change sharing to "Anyone with the link can view"
4. Update the download links in `index.html` to point to the Google Drive link
